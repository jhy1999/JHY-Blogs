num中-S,-g,-D的区别

npm install module_name -S 即 npm install module_name –save 写入 dependencies（开发环境）

npm install module_name -D 即 npm install module_name –save-dev 写入 devDependencies(生产环境)

npm install module_name -g 全局安装(命令行使用)

npm install module_name 本地安装(将安装包放在 ./node_modules 下)
