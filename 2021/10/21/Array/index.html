数组的深复制与浅复制

当我们把数组赋给另外一个数组，然后改变其中一个数组的值，另一个数组也会随之改变，这就是数组的浅复制。而深复制指的是不改变原数组而去创建一个新数组，这种情况是经常使用的，为了不破坏原数组。

存取函数

javaScript 提供了一组用来访问数组元素的函数，叫存取函数。最常用的存取函数就是indexOf()函数，该函数返回指定查找的值在目标值中是否存在，返回该值在数组中的索引，不存在则返回-1。

除此之外，还有join()和toString()函数，concat()和splice()函数。前两个函数可以将数组转化为字符串，后面两个函数可以通过已有的数组创建新数组，其中concat()方法通过合并多个数组来形成新数组，而 splice 方法是截取一个数组的子集作为一个新数组。

可变函数

不去引用数组的某个元素，就能改变数组内容，这种函数称之为可变函数。

push()和unshift()、pop()和shift()

push()方法可以在数组末尾添加元素，而 unshift()方法可在数组开头添加元素；相对应的，pop()可以删除数组末尾的元素，而 shift()删除数组第一个元素。

splice()、sort()、reverse()

splice()不仅可以删除元素，还可以用来添加元素。用sort()可以为数组排序，reverse()将数组内元素翻转。

关于 sort() 方法非常有意思，它只能对那些字符串类型的元素排列得比较准确，但如果是数字，结果就不那么令人满意了。

二维数组

JavaScript 可以通过在数组里再嵌套一个数组形成二维数组。

对于二维数组的处理可以分为两种，一种按列访问，一种是按行访问。 按列访问，外层循环对应行，内层循环对应列。例如，上述的数组，每一行对应一个学生三门科目的成绩记录，可以通过相加所有成绩，然后除以科目数来得到该生的平均成绩。
